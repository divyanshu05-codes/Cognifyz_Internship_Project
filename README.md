## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/divyanshu05-codes/Cognifyz_Internship_Project.git
```

Navigate to the project:

```bash
cd Cognifyz_Internship_Project
```

Open any task folder and install dependencies (if applicable):

```bash
npm install
```

Run the project:

```bash
npm start
```

or

```bash
npm run dev
```
